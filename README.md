# 📊 Sales Dashboard – Power BI

An interactive sales analytics dashboard built with **Power BI**, 
covering four years of order data (2015–2018) across multiple 
European regions.

## 📁 Dataset Overview

| File | Description |
|------|-------------|
| `order_2015.csv` – `order_2018.csv` | Yearly order-level data including sales, profit, discount, quantity, category, and shipping details |
| `Target.csv` | Annual sales targets by product category (Furniture, Technology, Office Supplies) |
| `Returns.csv` | Records of returned orders |
| `People.csv` | Regional managers mapped to North, South, and Central regions |
| `Rahul_Sales_Dashboard.pbix` | Power BI report file with all visuals and data model |

## 🔍 Key Insights

- Sales and profit trends across **2015–2018**
- Performance by **category**: Furniture, Technology, Office Supplies
- **Regional breakdowns** with assigned managers
- **Return rate analysis** by order
- Actual sales vs. **targets** by year and category

## 🛠️ Tools Used

- **Power BI Desktop** – Data modeling & visualization
- **CSV files** – Raw data sources

## 🚀 How to Use

1. Clone this repository
2. Open `Rahul_Sales_Dashboard.pbix` in Power BI Desktop
3. Ensure the CSV files are in the same directory as the `.pbix` file
4. Refresh the data source if prompted
